# 중첩 반복문과 전개 연산자

## 중첩 반복문
```python
# 리스트
a = [1,2,3]

# 2차원 리스트
a = [[1,2,3],[4,5,6],[7,8,9]]
```

## 전개 연산자

```python
# *리스트 -> 리스트 전개 연산자가 되어서 리스트를 한개 씩 분리하게 된다.
# 요소, 요소, 요소
## (1) 리스트 내부
a = [1,2,3]
b = [*a, *a]  # [1,2,3,1,2,3]

### 파괴적 연산 
a = [1,2,3]
a.append(4)  # [1,2,3,4]

### 비파괴적 연산
a = [1,2,3]  # a가 유지된다.
b = [*a, 4]  # [1,2,3,4]

## (2) 함수의 매개변수 위치
date = [2022, 10, 26, 11, 11]

# "{}년 {}월 {}일 {}시 {}분".format(date[0], date[1], date[2], date[3], date[4]) 이것보다
"{}년 {}월 {}일 {}시 {}분".format(*date) # 함수의 매개변수 위치에 전개 연산자를 활용하여 매개변수를 풀어서 출력할 수 있다.


```
