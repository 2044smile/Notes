# 리스트 함수

# 파괴적이다: 연산 후에도 피연산자가 번형되지 않는 것
## = 할당 연산자
```python
a = 10
print(a) # a = 10

a = 20
print(a) # a = 20
```

# 비파괴적이다: 연산 후에도 피연산자가 변형되지 않는 것
## +-*/ 연산자
## 장점: 원본과 결과가 모두 남는다. -> 안전하다.
## 단점: 원본과 결과가 둘 다 남긴다. -> 메모리를 많이 차지한다.
```python
a = 10
print(a) # a = 10
a + 20
print(a) # a = 10
b = a + 20
print(b) # b = 30
```
