## 마트에서 어떤 제품 관리
```python
product = [
    ['건망고 슬라이스', 4000, '식품'],
    ['인스타360 링크', 140000],
    ['와콤 프로펜', 15000]
]

product[0][1]
```

## 딕셔너리 탄생
```python
dic = {
    '제품명': 'cslee',
    '가격': '100000000000000',
    '분류': 'human'
}

dic['제품명']  # cslee
dic['가격']  # 4000
dic['분류']  # human


```
