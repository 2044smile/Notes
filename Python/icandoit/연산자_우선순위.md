# 연산자 우선순위

프로그래밍은 수학과 같이 수식이 직선으로 적혀있어도 뒤에 것이 먼저 계산되는 경우가 있다.
```python
2 + 2 * 2 = 6
```

## () 괄호 연산자
(): 계산이 먼저 일어나게 하는 기능

연산자 우선순위가 있음에도 적어주는 것이 좋다.
```python
(2 + 2) * 2
= 4 * 2
= 8
```

## 제곱근

무엇을 제곱했을 때, 이게 나오느냐? 2를 제곱했을 때 4가 나온다.
```python
> 4 ** (1/2)
2.0
> 8 ** (1/3)
2.0
> 16 ** (1/4)
2.0
```
